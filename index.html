🏠 About Yoran Bot

Yoran Bot is a Discord bot developed for Roblox complex systems shop servers. It provides server management tools, a support ticket system, automated role management, community utilities, and YouTube subscription verification to grant access to exclusive content.

This bot is deployed on private Discord servers and is not a public application available to all users. Its purpose is to enhance the moderation and purchasing experience within a specific Roblox shop community.

YouTube Verification: Yoran Bot uses Google OAuth 2.0 (Device Authorization Flow) to verify that a Discord user is subscribed to our YouTube channel. This is the only time user data from Google is accessed. We only request read-only access to subscription data and do not store any tokens or personal information.

🔐 Data & Privacy

Yoran Bot is designed with privacy in mind. The only external API we interact with is Google's YouTube Data API v3, and only when a user voluntarily initiates the /verifyyoutube command.

  • We request only youtube.readonly scope — the minimum necessary
  • OAuth access tokens are used once and never stored
  • No personal data is sold, shared, or used for advertising
  • Users can revoke access at any time via their Google account
